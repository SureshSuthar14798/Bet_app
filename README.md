# NexusBet Premium Sportsbook

A modern, high-performance sports betting application built with Next.js 15 and React 19. Designed for a premium user experience with real-time updates, sleek animations, and comprehensive account management features.

## ğŸ“¦ Getting Started

### Prerequisites

- Node.js (Latest LTS version recommended)
- npm or yarn

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/SureshSuthar14798/Bet_app.git
   cd Bet_app
   ```

2. **Install dependencies:**
   ```bash
   npm install
   # or
   yarn install
   ```

3. **Run the development server:**
   ```bash
   npm run dev
   # or
   yarn dev
   ```

4. **Open your browser:**
   Navigate to [http://localhost:3000](http://localhost:3000) to see the application running.
